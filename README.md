# ExfilZone Tracker

A comprehensive web-based companion tool for **ExfilZone** (Contractors VR DLC). Track your hideout upgrades, view ammo statistics, find key vault locations, and more.

**Live Site:** https://yagercodes.github.io/ExfilZoneTracker/

## Features

### Hideout Tracker
- Track progress on all hideout upgrades with a visual card-based interface
- **Items Needed** - Collapsible panel showing all items required for remaining upgrades
- **Up Next** - Shows only the immediate next upgrade for each category (focus on what you need now, not everything)
- DLC ownership toggle - Automatically marks upgrades unlocked by NTG Standard Issue or Johnny's Secret Stash DLCs
- Search and filter upgrades by name or items
- Progress saved locally in your browser

### Ammo TTK Chart
- Time-to-kill data for all ammunition types
- Filter by caliber and armor class
- Visual damage indicators

### Key Vault Guide
- Vault locations for Smuggling Tunnel, Suburbs, and Resort maps
- Notable loot information including rare spawns
- Average value estimates for planning runs

### Key Spawn Guide
- World spawn locations for keys
- Map-specific spawn points

### Interactive Maps
- Quick links to Showdown Wiki interactive maps
- Coverage for Suburbs, Metro, Dam, and Resort

## Tech Stack

Pure HTML, CSS, and JavaScript - no build tools or frameworks required. Just open the HTML files in a browser.
